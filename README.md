# 클론코딩- 주문 관리 시스템

## 프로젝트

카페 `Grids&Circles` 의 온라인 웹사이트 주문 사이트입니다.

### 요구사항

---

- 관리자

  + 상품 추가 

- 고객(주문 관리 페이지)


  + Email, 주소, 이름, 우편번호를 입력후 주문

    (Email로 고객을 식별한다.)
